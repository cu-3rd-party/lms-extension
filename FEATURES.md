# Полный список функций расширения CU LMS Enhancer

Расширение работает на сайте `my.centraluniversity.ru` и добавляет следующие возможности, сгруппированные по области применения.

---

## Глобальные (все страницы LMS)

| Функция                     | Включение             | Описание                                                                  |
| --------------------------- | --------------------- | ------------------------------------------------------------------------- |
| **Тёмная тема**             | Попап → «Тёмная тема» | Полная тёмная тема для всего LMS, включая Shadow DOM компоненты           |
| **OLED-тема**               | Попап → «OLED»        | Абсолютно чёрный фон для OLED-экранов (расширение тёмной темы)            |
| **Кнопка меню расширения**  | Всегда включена       | Добавляет кнопку в шапку сайта для быстрого доступа к настройкам          |
| **Вкладка «Друзья»**        | Попап → «Друзья»      | Новая вкладка в навигации — список одногруппников с отметкой общих курсов |
| **Снег**                    | Попап → «Снег»        | Анимация падающих снежинок поверх страницы (праздничная фишка)            |
| **Кастомные иконки курсов** | Попап → «Стикеры»     | Замена стандартных иконок карточек курсов на любые картинки/стикеры       |
| **Проверка версии**         | Всегда включена       | Уведомляет в шапке, если установленная версия расширения устарела         |
| **Сброс данных LMS**        | Кнопка в меню         | Очищает кэш фильтров и список друзей из localStorage                      |
| **Фикс цвета иконок**       | Всегда включен        | Исправляет отображение SVG-иконок при нестандартных темах                 |

## Страница списка курсов (`/learn/courses/view`)

| Функция                         | Включение                      | Описание                                                                                                  |
| ------------------------------- | ------------------------------ | --------------------------------------------------------------------------------------------------------- |
| **Упрощённый дизайн карточек**  | Попап → «Старый дизайн курсов» | Заменяет объёмные карточки курсов на компактные текстовые плашки                                          |
| **Архивация курсов**            | Иконка на карточке             | Скрывает ненужные курсы в архив; заархивированные доступны во вкладке «Archived»                          |
| **Перетаскивание курсов**       | Активен при старом дизайне     | Drag-and-drop сортировка карточек курсов                                                                  |
| **Статус задач в обзоре курса** | Попап → «Статус задач»         | В аккордеоне курса рядом с каждой темой показывает статус задания (сдано / просрочено / скип / доработка) |
| **Автоскролл к текущей неделе** | Попап → «Автоскролл»           | При открытии обзора курса автоматически разворачивает и прокручивает к последней (актуальной) теме        |
| **Виджет «Друзья на курсе»**    | Попап → «Друзья»               | Плавающий виджет на странице курса — показывает, кто из друзей записан на тот же курс                     |
| **Предстоящие контрольные**     | Попап → «Расписание КР»        | Отображает в аккордеоне курса даты будущих контрольных и экзаменов                                        |

## Страница задач (`/learn/tasks`)

| Функция                        | Включение              | Описание                                                                                |
| ------------------------------ | ---------------------- | --------------------------------------------------------------------------------------- |
| **Колонка «Вес задания»**      | Всегда включена        | Добавляет в таблицу задач колонку с весом каждого задания в итоговой оценке             |
| **«Метод скипа»**              | Кнопка в строке задачи | Помечает задание как намеренно пропущенное; отображается кастомным чипом                |
| **Выделение семинаров**        | Всегда включена        | Строки с семинарными заданиями окрашиваются в отдельный цвет для наглядности            |
| **Пользовательские фильтры**   | Поле поиска в таблице  | Фильтрация задач по имени, курсу, статусу прямо в таблице                               |
| **Сброс стандартных фильтров** | Автоматически          | При загрузке страницы сбрасывает стандартные фильтры LMS, чтобы показывались все задачи |
| **Замена эмодзи в задачах**    | Попап → «Сердечки»     | Заменяет круглые эмодзи (🔵🔴⚫) на сердца (💙❤️🖤) прямо в именах задач                |

## Страница задания (`/longreads/...`)

| Функция                                | Включение               | Описание                                                                                                 |
| -------------------------------------- | ----------------------- | -------------------------------------------------------------------------------------------------------- |
| **Вес задания**                        | Всегда включена         | Добавляет в метаинформацию задания его вес в итоговой оценке курса (% от финального балла)               |
| **Просмотр документа в модалке**       | Кнопка рядом с файлом   | Открывает вложенный документ/изображение в модальном окне без перехода на отдельную страницу             |
| **Автопереименование ДЗ**              | Попап → «Авторенейм ДЗ» | Автоматически переименовывает загружаемый файл по шаблону (ФИО_неделяN, dz_fi и др.) перед отправкой     |
| **Статус «Метод скипа» / «Доработка»** | Связан с tasks_fix      | Отображает рядом с заданием его особый статус, если задание помечено как скипнутое или требует доработки |

## Страница успеваемости (`/learn/reports/student-performance`)

| Функция                          | Включение                       | Описание                                                                                                   |
| -------------------------------- | ------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| **Расширенная ведомость**        | Попап → «Расширенные ведомости» | Добавляет детальную разбивку итоговой оценки по категориям (ДЗ, КР, экзамен) с весами и накопленным баллом |
| **Прогноз итоговой оценки**      | Часть расширенной ведомости     | Рассчитывает, какой балл нужно набрать на оставшихся работах, чтобы получить желаемую оценку               |
| **Архивация курсов в ведомости** | Иконка на строке курса          | Скрывает завершённые курсы из основной ведомости; архивные курсы доступны во вкладке архива                |
| **Калькулятор GPA**              | Всегда включён на странице      | Отображает средний балл (GPA) по всем курсам, отдельно для активных и архивных                             |

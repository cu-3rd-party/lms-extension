# Плагин `dark-theme` — тёмная тема

**URL:** все страницы `https://my.centraluniversity.ru/`

Полная тёмная тема для LMS с поддержкой Shadow DOM компонентов и опциональным OLED-режимом.

---

## Файлы и их назначение

### `index.loader.js`

Скрипт-загрузчик темы. Является ES-модулем (в отличие от остальных плагинов — plain JS), потому что использует `import` для инлайн-бандлинга CSS через Vite.

**Что делает:**

1. Читает настройки `themeEnabled` и `oledEnabled` из `browser.storage.sync`.
2. При включении темы инжектирует `<style>` с базовым тёмным CSS (`dark-theme.css`) через функцию `insertCss`.
3. Если включён OLED-режим — дополнительно инжектирует `dark-theme-oled.css` с абсолютно чёрным фоном.
4. CSS для Shadow DOM (`shadow-dom-dark.css`, `shadow-dom-oled.css`) применяется через `browser.tabs.insertCSS` / спец. обёртку для компонентов в closed Shadow DOM.
5. Добавляет кнопку переключения темы (☀️/🌙) в шапку сайта.
6. Подписывается на `browser.storage.onChanged` — реагирует на изменение `themeEnabled` / `oledEnabled` мгновенно, без перезагрузки страницы.
7. Исправляет цвета иконок уровней в карточках курсов (`cu-course-card .skill-level`), которые зашиты инлайн и не перекрываются CSS-переменными.

**Взаимодействие со страницей:**

- Вставляет: `<style>` теги с тёмной темой в `<head>`
- Читает DOM: `ul.header__actions-list` для кнопки переключения
- Не модифицирует структуру страницы

---

### `dark-theme.css`

Основная тёмная тема. Переопределяет CSS-переменные и стили компонентов TaigaUI, используемых в LMS (`--tui-base-*`, `--tui-text-*`, `--tui-primary` и т.д.), а также кастомные элементы LMS.

Содержит точечные фиксы для элементов, которые LMS красит инлайном или через псевдоэлементы, например help-иконок в `cu-tooltip` на странице формулы оценивания. Для них цвет переопределяется и через `tui-icon::after`, и через `--cu-tooltip-icon-color`, а для `tui-icon.hint` дополнительно форсируется светлый рендер через `filter`, потому что один computed color не гарантирует светлые пиксели у SVG-иконки.
Сохраняет индивидуальные обложки карточек курсов, если LMS прокидывает их через `--cu-island-cover-bg`, чтобы тёмная тема не перекрашивала такие карточки в общий фон.
Отдельно содержит точечные фиксы для интерактивных элементов LMS, например hover-состояния строк задач, чтобы встроенные фильтры темы не ломали ни кастомные action-иконки расширения, ни штатные `warning-icon` внутри строки.

---

### `dark-theme-oled.css`

Расширение тёмной темы для OLED-экранов. Переопределяет фоновые цвета до полностью чёрного (`#000000`) для минимального потребления энергии.

---

### `shadow-dom-dark.css` / `shadow-dom-oled.css`

CSS, который применяется внутри Shadow DOM компонентов (закрытых web-компонентов TaigaUI). Имеет ту же логику, но для теневого дерева, недоступного через обычный `document.querySelector`.

---

## Архитектурная особенность

Этот плагин — единственный, который собирается Vite как ES-модуль (`?script` импорт в манифесте). CSS-файлы встраиваются в JS как строки (`?inline`) при сборке, поэтому не требуют отдельного fetch-запроса в рантайме.

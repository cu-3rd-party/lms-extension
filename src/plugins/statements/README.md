# Плагин `statements` — страница успеваемости

**URL:** `/learn/reports/student-performance` (ведомость успеваемости)

Набор скриптов для расширения функциональности страницы успеваемости: детальная разбивка оценок, архивация курсов и подсчёт GPA.

---

## Файлы и их назначение

### `advanced_statements.js`

> Активируется только при включённом `advancedStatementsEnabled` в настройках.

Добавляет детальную разбивку итоговой оценки по категориям работ.

**Как работает:**

1. Срабатывает на URL-паттерн `/learn/reports/student-performance/.../activity` (страница активности по конкретному курсу).
2. Запрашивает данные двух API:
   - `/api/micro-lms/courses/{courseId}/student-performance` — результаты студента
   - `/api/micro-lms/courses/{courseId}/exercises` — список упражнений с категориями
3. Обогащает данные из встроенной базы `COURSE_DATA` — объект с весами категорий (ДЗ, КР, коллоквиум, экзамен и т.д.) для конкретных курсов.
4. Под стандартной таблицей вставляет новый блок с:
   - разбивкой оценки по категориям (балл / максимум / вес)
   - накопленным текущим баллом
   - прогнозом: сколько нужно набрать на оставшихся работах для желаемой итоговой оценки
5. Если включена опция `endOfCourseCalcEnabled` — показывает калькулятор итоговой оценки.

**Взаимодействие со страницей:**

- Читает DOM: стандартную таблицу успеваемости
- Вставляет: дополнительный блок под таблицей
- Fetch: 2 API-запроса к LMS

### `archive-statements.js`

Добавляет возможность архивировать курсы в ведомости успеваемости.

**Как работает:**

1. Срабатывает на всех страницах `/learn/reports/student-performance`.
2. Для каждой строки таблицы курсов добавляет кнопку-архиватор (иконка 📦).
3. При клике:
   - Скрывает строку курса (`display: none`).
   - Сохраняет данные курса (href, название, балл) в `browser.storage.local` (`cu.lms.archived-statements`).
4. При загрузке страницы читает архив и применяет скрытие к уже заархивированным курсам.
5. Добавляет вкладку «Архив» в навигацию через `tui-breadcrumbs`, где показываются скрытые курсы с возможностью разархивировать каждый.
6. При изменении темы (`themeEnabled`) обновляет цвет иконки архива.

**Взаимодействие со страницей:**

- Читает DOM: `table.cu-table`, `tr[tuitr]`, `tui-breadcrumbs`
- Вставляет: кнопки архиватора, вкладку «Архив», строки разархивированных курсов
- Хранилище: `browser.storage.local` (cu.lms.archived-statements)

### `metrics_statements.js`

Рассчитывает и отображает GPA (средний балл) студента.

**Как работает:**

1. Срабатывает на любой подстранице `/learn/reports/student-performance`.
2. Запрашивает `/api/micro-lms/performance/student` — список всех курсов с итоговыми баллами.
3. Читает из `browser.storage.local` список заархивированных курсов.
4. Фильтрует служебные курсы (вступительные контесты, пульс-опросы, ознакомительные).
5. Рассчитывает GPA как среднее арифметическое итоговых баллов.
6. Вставляет на страницу таблицу с тремя значениями GPA:
   - По всем курсам
   - Только по активным (не заархивированным)
   - Только по заархивированным
7. Подписывается на `browser.storage.onChanged` — при изменении архива пересчитывает GPA без перезагрузки.

**Взаимодействие со страницей:**

- Вставляет: `<div id="lms-gpa-table-container">` с таблицей GPA
- Fetch: `/api/micro-lms/performance/student`
- Хранилище: `browser.storage.local` (cu.lms.archived-statements)

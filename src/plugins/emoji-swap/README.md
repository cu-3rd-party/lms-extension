# Плагин `emoji-swap` — замена эмодзи

**URL:** все страницы LMS, **кроме** `/learn/tasks`

> На странице `/learn/tasks` используется встроенная версия этой логики в `courses/tasks_fix.js`, чтобы избежать конфликта двух скриптов.

Заменяет круглые цветные эмодзи (🔵🔴⚫) в именах задач и курсов на сердца (💙❤️🖤).

---

## Файлы и их назначение

### `emoji_swap.js`

**Как работает:**

1. Читает флаг `emojiHeartsEnabled` из `browser.storage.sync`.
2. Если включено — обходит всё дерево DOM с помощью `TreeWalker` (только текстовые узлы) и заменяет:
   - `🔵` → `💙`
   - `🔴` → `❤️`
   - `⚫️` / `⚫` → `🖤`
3. Также обходит Shadow DOM: для каждого элемента с `shadowRoot` проходит по его дереву.
4. `MutationObserver` следит за изменениями в DOM и повторяет замену для новых узлов (SPA-навигация, динамическая подгрузка).
5. При выключении настройки делает обратную замену (сердца → кружки).

**Координация с `tasks_fix.js`:**

- Скрипт ждёт сигнала от `tasks_fix.js` через кастомное событие `window` перед первым запуском.
- Если сигнал не пришёл за 2 секунды — считает, что страница не задачная и запускается самостоятельно.

**Взаимодействие со страницей:**

- Только изменение `nodeValue` текстовых узлов DOM — структуру страницы не меняет
- Хранилище: `browser.storage.sync` (emojiHeartsEnabled)
